# RN Wallpaper

RN-Wallpaper is a React Native app that allows you to browse, search and download high-quality wallpapers from Unsplash's vast collection. With a simple and intuitive user interface, you can easily set your favorite wallpapers as your device's background and enjoy stunning images every day.


# Features

## Search images

Users can search for specific images by keyword or phrase.
## View trending images
Users can view a collection of popular images from Unsplash.
## Search any image
The app allows users to search for any image on Unsplash, giving them access to a vast collection of high quality images.
Download images: Users can download images from Unsplash to their device for offline viewing or to use in their own projects.

## Save to local storage 
The app allows users to save images to their device's local storage for easy access in the future.
